# Débuter avec l'application Magic Kebab

Projet propulsé par <b>Create React App</b> et <b>Bootstrap</b>.

<h3> Features disponibles:</h3>

<ul>
    <li>Création et configuration du kebab</li>
    <li>Choix d'avoir un kebab végétarien</li>
    <li>Choix des crudités</li>
    <li>Choix des sauces (2 max)</li>
    <li>Commande possible</li>
    <li>Suppression, modification de la quantité dans le panier</li>
    <li>Menu existant avec 3 kebabs</li>
    <li>Possibilité de retourner aux étapes de configuration précédentes (si vous faites une erreur)</li>
    <li>Calcul automatique du prix avec la quantité et la carte</li>
</ul>

## Script disponibles

Dans le projet vous pouvez démarrer l'application avec:

### `npm start`

Cela va lancer l'application sur le port par defaut.\
Ouvrez [http://localhost:3000](http://localhost:3000) pour voir le résultat.

### `npm run build`

Build la version de production de l'application dans le dossier `build` .\

La construction est minifiée et les noms de fichiers incluent les hash.\
L'application est ensuite prête à être déployée!
